5. KEY SECURITY FINDINGS DOCUMENTED
------------------------------------

  Current:
    - RBAC with 5 role levels, module permissions, and user-level overrides
    - AES-256-CBC encryption for vault passwords and SMTP credentials
    - Login audit trail (success, failure, suspension, logout)
    - Session regeneration on login/logout + CSRF protection + rate limiting
    - Suspended user enforcement at middleware level (web + API)
    - Activity logging with sensitive field exclusions
    - RBAC-aware search with 4 ownership scoping modes
    - CSV export with RBAC gating and data scoping

  Gaps (v1.0 known limitations):
    - No multi-factor authentication
    - No password complexity rules (min 8 only)
    - No account lockout (rate limiting per-minute only)
    - No IP whitelisting for admin routes
    - No file upload malware scanning
    - No security headers (CSP, HSTS, X-Frame-Options not configured)
    - CSV-only export (potential formula injection)
