# DEPLOY EXCLUDE LIST

> Files and directories that SHOULD NOT be uploaded to production.
> Use this as a deploy script exclusion filter (rsync --exclude, .dockerignore, deploy.sh skip patterns).

---

## Tier 1: MUST EXCLUDE (secrets, config, runtime-only)

| Pattern | Reason |
|---|---|
| `.env` | Database credentials, APP_KEY, secrets |
| `storage/logs/*.log` | Runtime logs — recreated by Laravel |
| `storage/framework/cache/*` | Runtime cache |
| `storage/framework/sessions/*` | Active session files |
| `storage/framework/views/*` | Compiled Blade cache — regenerated by `view:cache` |
| `bootstrap/cache/*.php` | Config/service cache — regenerated by `optimize` |
| `storage/api-docs/api-docs.json` | Generated Swagger doc |

---

## Tier 2: SHOULD EXCLUDE (development-only assets)

| Pattern | Reason | Est. Size |
|---|---|---|
| `_can_delete/` | Quarantine — reports, logs, cache, backups | 74 MB |
| `coverage/` | HTML code coverage report (if restored) | 17 MB |
| `tests/coverage/` | HTML coverage (if restored) | 7 MB |
| `tests/coverage-xml/` | XML coverage reports | 7 MB |
| `node_modules/` | npm dev dependencies (unless CI builds on server) | 69 MB |
| `.phpunit.result.cache` | PHPUnit result cache | 220 KB |
| `phpunit.xml` | Test configuration | — |
| `phpstan*.neon` | PHPStan configs | 20 KB |
| `Dockerfile` | Docker image build | — |
| `docker-compose.yml` | Docker compose local dev | — |
| `.dockerignore` | Docker context rules | — |
| `deploy.sh` | Deployment script | — |
| `.env.e2e` | E2E test environment | — |
| `.env.testing` | Testing environment | — |
| `.editorconfig` | IDE/editor configuration | — |
| `.phpactor.json` | PHP actor config (if present) | — |

---

## Tier 3: CONSIDER EXCLUDING (debatable)

| Pattern | Reason | Decision |
|---|---|---|
| `tests/` (source tests) | Not needed at runtime | **Exclude** unless running tests on production |
| `resources/js/` | Source JS — built by Vite | **Exclude** if `public/build/` is deployed |
| `resources/css/` | Source CSS — built by Vite | **Exclude** if `public/build/` is deployed |
| `docs/` | Project documentation | **Exclude** |
| `scripts/` (if restored) | Dev/test helper scripts | **Exclude** |
| `e2e/` (if restored) | End-to-end test fixtures | **Exclude** |
| `.github/` | CI/CD workflows | **Exclude** |
| `deploy/` (if restored) | Deployment reference | **Exclude** |

---

## Deploy Script Exclusion Patterns

```bash
# rsync --exclude patterns
--exclude='.env'
--exclude='_can_delete'
--exclude='coverage'
--exclude='tests/coverage'
--exclude='tests/coverage-xml'
--exclude='node_modules'
--exclude='storage/logs/*.log'
--exclude='storage/framework/cache/*'
--exclude='storage/framework/sessions/*'
--exclude='storage/framework/views/*'
--exclude='storage/api-docs'
--exclude='bootstrap/cache/*.php'
--exclude='.phpunit*'
--exclude='phpunit*'
--exclude='phpstan*'
--exclude='Dockerfile*'
--exclude='docker-compose*'
--exclude='.dockerignore'
--exclude='deploy.sh'
--exclude='.env.e2e'
--exclude='.env.testing'
--exclude='.editorconfig'
--exclude='docs/'
--exclude='scripts/'
--exclude='e2e/'
--exclude='.github/'
--exclude='resources/js/'
--exclude='resources/css/'
```
