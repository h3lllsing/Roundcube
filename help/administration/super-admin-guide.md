# Super Admin Guide

> **Audience:** Super Administrators — **Purpose:** Configure and maintain the OpsPilot portal

## Overview

This guide covers administrative functions available only to Super Administrators.

## User Management

Create, edit, suspend, and delete user accounts. Assign roles and manage module-level permissions.

## Role Management

Define custom roles with specific permission sets. Clone existing roles as templates for new configurations.

## Module Permissions

Configure which roles can Access, Manage, Import, and Export data in each module.

## Security

- **Credential Reveal** — Control who can view plain-text passwords in the Vault
- **Activity Logs** — Review all user actions across the portal
- **Login Audits** — Monitor authentication attempts and failed logins

## System Configuration

- SMTP profiles for email delivery
- Webhook endpoints for integrations
- API token management
- System-wide notification settings

## Delete & Restore

Super Administrators can permanently delete or restore business records. These actions are irreversible for deletions.
