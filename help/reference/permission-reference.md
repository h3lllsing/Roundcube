# Permission Reference

> **Audience:** All Users — **Purpose:** Understand the permission system and role capabilities

## Overview

OpsPilot uses a role-based access control (RBAC) system. Each user is assigned a role that determines what they can see and do.

## Role Levels

| Role | Description |
|------|-------------|
| Super Admin | Full access to all modules, settings, and administrative functions |
| Admin | Manage business records, tasks, and operational data |
| IT Support | View and update assigned records, manage personal tasks |
| Read-Only | View records and dashboards without making changes |

## Module Permissions

Each module supports four permission levels:

- **Access** — View records in the module
- **Manage** — Create, update, and delete records
- **Import** — Bulk import records from external sources
- **Export** — Export module data

## Permission-Dependent Documentation

Documentation in this Help Center adapts to your role. Features you do not have permission to use are marked accordingly.
