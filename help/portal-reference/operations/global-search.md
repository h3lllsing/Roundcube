## Overview

The Global Search lets you search across all portal modules from a single input field. Results are scoped to records you have **Access** to.

## Permission Controls

| Control | Description |
|---------|-------------|
| Access | Use the global search |

No other permissions are required — search results respect the user's existing Access permissions per module.

## How It Works

1. **Trigger**: Click the search icon in the top navigation bar, or press Ctrl+K / Cmd+K.
2. **Query**: Start typing — results appear as you type.
3. **Modules Searched**: All modules the user can access, including Domains, Hostings, VPS, VoIP, Other Services, Assets, Notes, Tasks, and more.
4. **Results**: Each result shows the module icon, record title, and a preview snippet. Click to navigate directly to the record.
