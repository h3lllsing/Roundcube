## Overview

The Notes module lets users create internal notes for personal or shared reference. Notes support rich content and can be linked to other portal records.

## Permission Controls

| Control | Description |
|---------|-------------|
| Access | View and list notes |
| Manage | Create, edit, and delete notes |

## Features

### Note Properties
- Title and content
- Category or tag labels
- Pinned / unpinned status

### Linking
Notes can reference other portal records (e.g., "Deployment notes for VPS xyz"). The association is displayed when viewing the linked record.
