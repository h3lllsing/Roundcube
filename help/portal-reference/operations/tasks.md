## Overview

The Tasks module provides a simple to-do system for tracking work items. Tasks can be assigned, prioritised, and organised by status.

## Permission Controls

| Control | Description |
|---------|-------------|
| Access | View and list tasks |
| Manage | Create, edit, and delete tasks |

## Features

### Task Properties
- Title and description
- Priority (low, medium, high, urgent)
- Status (pending, in progress, completed, cancelled)
- Due date
- Assigned user

### Filtering & Search
Filter tasks by status, priority, assignee, or due date. Use the global search to find tasks by title.
