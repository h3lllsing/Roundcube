## Overview

The Calendar module displays scheduled events and upcoming deadlines from across the portal. It aggregates expiry dates, task due dates, and manually created events.

## Permission Controls

| Control | Description |
|---------|-------------|
| Access | View the calendar |
| Manage | Create, edit, and delete calendar events |

## Features

### Aggregated View
The calendar pulls in:
- Task due dates
- Expiry tracker deadlines
- Manually created events

### Event Types
- Manual events (meetings, reminders)
- Auto-synced deadlines from expiring records

### Date Navigation
Switch between month, week, and day views. Jump to specific dates.
