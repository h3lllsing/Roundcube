# Domains

> **Audience:** All Users — **Purpose:** Manage domain registrations and DNS records

## Overview

The Domains module lets you track domain registrations, expiry dates, DNS settings, and associated hosting accounts.

## Key Features

- View all domains in a sortable table
- Add new domain records with registrar and expiry information
- Link domains to hosting accounts and service providers
- Receive renewal reminders before expiry

## Permission-Dependent Features

- **Access** — View domain records
- **Manage** — Create, update, and delete domain records
- **Import** — Bulk import domains from CSV
- **Export** — Export domain data
