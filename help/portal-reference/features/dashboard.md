# Dashboard

> **Audience:** All Users — **Purpose:** Understand the Dashboard layout and available widgets

## Overview

The Dashboard is the first page you see after logging in. It provides a summary of your portal activity, upcoming renewals, and key metrics.

## Widgets

- **Summary Cards** — Total domains, hostings, VPS accounts, and other services
- **Upcoming Renewals** — Services expiring within the next 30 days
- **Recent Activity** — Latest changes across your managed modules
- **Quick Actions** — One-click access to common tasks
