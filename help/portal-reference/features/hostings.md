# Hostings

> **Audience:** All Users — **Purpose:** Manage web hosting accounts and server details

## Overview

The Hostings module centralises all web hosting account information including server credentials, resource limits, and renewal dates.

## Key Features

- Track hosting plans, providers, and pricing
- Monitor disk usage and bandwidth limits
- Link hosting to domains and VPS accounts
- Set automated renewal reminders

## Permission-Dependent Features

- **Access** — View hosting records
- **Manage** — Create, update, and delete hosting records
- **Import** — Bulk import hosting records from CSV
- **Export** — Export hosting data
