## Overview

The Modules page (Super Admin only) lists every registered module in the portal. It serves as a registry reference for understanding which modules are available, importable, and exportable.

## Permission Controls

| Control | Description |
|---------|-------------|
| Access | View the modules registry |

> **Note**: The Modules page is restricted to Super Admins.

## Features

### Module Registry
Each module entry shows:
- Module name and slug
- Whether it supports Import and Export
- Enabled/disabled status

### Importable & Exportable Modules
The registry defines which modules support CSV import and export. These assignments determine which modules appear in the Import and Export screens.
