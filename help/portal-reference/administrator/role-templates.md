## Overview

Role Templates provide pre-configured permission sets that serve as starting points when creating new roles. Templates reduce configuration time and ensure consistent permission baselines.

## How Templates Work

When creating a new role, select a template to pre-populate all module permissions. After creation, permissions can be customised per-role without affecting the template.

## Available Templates

- **Viewer**: Access to all modules, no Manage or Reveal
- **Operator**: Access + Manage for infrastructure modules
- **Administrator**: Access + Manage + Reveal, plus admin privileges
- **Custom**: Start from a blank permission set
