## Overview

The Attachments module manages file uploads attached to portal records. Attachments are stored on the filesystem and linked to individual records.

## Permission Controls

| Control | Description |
|---------|-------------|
| Attachments | Access the Attachments screen |

The Attachments privilege controls access. Users also need Manage on the parent module to attach/remove files.

## Features

### Upload Limits
- Maximum file size: configured in system settings
- Allowed file types: images, PDFs, documents, archives
- Total storage limit: per organisation configuration

### Viewing Attachments
Attachments are displayed on the parent record's detail page. Clicking an attachment downloads the file.

### Management
Super Admins can view all attachments in a central list, sorted by record, module, or upload date.
