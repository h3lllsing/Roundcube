## Overview

Bulk Actions let you perform operations on multiple records at once — such as deleting, changing status, or updating providers across selected records.

## Permission Controls

| Control | Description |
|---------|-------------|
| Bulk Actions | Access bulk action tools |

The Bulk Actions privilege controls access. Users also need Manage permission on each affected module.

## Available Actions

| Action | Effect |
|--------|--------|
| Delete | Permanently deletes selected records |
| Change Status | Updates status field in bulk |
| Change Provider | Reassigns provider for selected records |
| Change Owner | Reassigns ownership |
