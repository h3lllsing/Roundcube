## Overview

The Reports module provides aggregated data views across portal modules. Reports help with capacity planning, cost analysis, and compliance reviews.

## Permission Controls

| Control | Description |
|---------|-------------|
| Reports | Access the Reports dashboard |

The Reports privilege controls access. Report contents respect the user's existing module permissions.

## Available Reports

| Report | Description |
|--------|-------------|
| Infrastructure Summary | Counts of all infrastructure records by type and status |
| Expiry Summary | Upcoming expirations grouped by time window |
| Cost Overview | Monthly/yearly costs aggregated by module |
| User Activity | Summary of user actions over a date range |

## Exporting Reports
Reports can be exported as CSV or PDF from the report view.
