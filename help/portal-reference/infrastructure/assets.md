## Overview

The Assets module tracks hardware, software licences, and digital assets owned or managed by your organisation.

## Permission Controls

| Control | Description |
|---------|-------------|
| Access | View and list assets |
| Manage | Create, edit, and delete assets |

## Features

### Asset Types
Categorise assets by type (hardware, software, licence, digital, other).

### Asset Details
- Name, type, serial/model number
- Purchase date, cost, warranty period
- Assigned user or location
- Status (active, retired, disposed)

### Document Attachments
Upload receipts, licence files, or warranty documents as asset attachments.

## Import / Export
- Import: CSV with columns for name, type, serial, purchase date, cost, status, notes
- Export: CSV download of all asset records
