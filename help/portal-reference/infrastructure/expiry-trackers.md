## Overview

The Expiry Trackers module monitors upcoming renewals and expirations across all infrastructure records. It aggregates expiry dates from Domains, Hostings, SSL certificates, and other dated services.

## Permission Controls

| Control | Description |
|---------|-------------|
| Access | View expiry tracker dashboard and list |
| Manage | Create, edit, and delete tracker records |

## Features

### Centralised Expiry View
See all upcoming expirations in one dashboard, sorted by date. Each tracker shows the associated item, its type, and days remaining.

### Notification Thresholds
Set reminder intervals (e.g., 30 days, 14 days, 7 days before expiry). The portal displays visual indicators for approaching and past-due dates.

### Status Tracking
Mark items as active, expiring, expired, or renewed.
