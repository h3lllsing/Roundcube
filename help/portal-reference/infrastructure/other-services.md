## Overview

The Other Services module records miscellaneous third-party services, subscriptions, and accounts that don't fit into dedicated modules (VPS, Domains, Hostings, etc.).

## Permission Controls

| Control | Description |
|---------|-------------|
| Access | View and list other services |
| Manage | Create, edit, and delete service records |

### Reveal
Some Other Service records store credentials. These fields require the Reveal capability on this module.

## Features

### Flexible Categorisation
Assign a service type label to group related services.

### Credential Storage
- Login URLs and account identifiers
- Username and password fields
- Reveal-required for credential display

### Provider & Cost Tracking
Link to a Service Provider and record billing details.

## Import / Export
- Import: CSV with columns for label, type, URL, provider, status, cost, notes
- Export: CSV download of all other service records
